Portfolio Tracker

Track and analyze your cryptocurrency investments

About This Application

Portfolio Tracker is a Spring Boot web application designed to help you monitor and manage your cryptocurrency investments. The application provides real-time tracking of your portfolio performance with a clean, intuitive interface.

Key Features:

  • Real-time cryptocurrency price tracking
  • Portfolio performance analytics
  • Interactive charts and visualizations
  • Responsive design for all devices
  • Secure data persistence with H2 database

Built With:

Spring Boot H2 Database Java HTML5 CSS3 JavaScript Bootstrap Docker
